# TP Bases de Datos 2019 - Angular

Este proyecto se divide en:
- **Client**
- **Server**

Para correrlo es preciso instalar todas las dependencias necesarias del cliente y el servidor y ponerlos a correr:
### Cliente:

```
cd client
npm install
ng s -o
```

### Server:
```
cd server
npm install
npm run build
```
En una nueva terminal:
```
cd server
npm run dev
```
